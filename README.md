Taste
=====

Installation
------------
```sh
$ git clone https://github.com/d9nny/taste
```
Navigate to root directory of the project:
```sh
$ bundle 
$ createdb taste
$ rake db:auto_migrate
```
To run locally:
```sh
$ rails s
```
Testing
-------
To run tests:
```sh
$ rspec
```
